List Experiments
================

A set of quick and dirty methods to benchmark the performance of various List implementations in Java.

Overview
--------
Here are a set of sorting algorithms and how long it takes to do 10000 iterations of each of them.

Usage
-----
To run this program, run the following commands in your terminal while being in the folder that holds the program:

javac Experiments.java
java Experiments
